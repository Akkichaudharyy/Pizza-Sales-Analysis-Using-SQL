# 🍕 Pizza Sales Analysis Using SQL

## Project Overview

This project analyzes pizza sales data using SQL to uncover valuable business insights related to customer behavior, sales performance, revenue trends, and product popularity. The analysis was performed using SQL queries ranging from basic data retrieval to advanced analytical queries involving joins, aggregate functions, and window functions.

## Objectives

* Analyze overall sales performance.
* Calculate total revenue generated.
* Identify top-selling and highest revenue-generating pizzas.
* Understand customer ordering patterns.
* Evaluate category-wise sales performance.
* Generate business recommendations based on data insights.

## Database Tables

* Orders
* Order_Details
* Pizzas
* Pizza_Types

## SQL Concepts Used

* SELECT Statements
* WHERE Clause
* GROUP BY
* ORDER BY
* LIMIT
* INNER JOIN
* Aggregate Functions (SUM, COUNT, AVG)
* Window Functions (RANK, OVER)

## Key Business Insights

* Total Orders: 21,350
* Total Revenue: $817,860.05
* Highest Priced Pizza: The Greek Pizza ($35.95)
* Most Ordered Size: Large (18,526 Orders)
* Top Revenue Pizza: The Thai Chicken Pizza ($43,434.25)
* Highest Revenue Category: Classic (26.91%)

## Advanced Analysis Performed

* Revenue Contribution Analysis
* Cumulative Revenue Trend Analysis
* Top Revenue-Generating Pizzas
* Category-wise Revenue Ranking
* Peak Ordering Hour Analysis

## Results

The analysis revealed that Large pizzas were the most popular among customers, while lunch hours recorded the highest number of orders. The Classic category generated the highest revenue contribution, and Thai Chicken Pizza emerged as the top revenue-generating product.
